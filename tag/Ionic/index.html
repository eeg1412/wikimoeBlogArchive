解决Ionic在iOS中无法长按保存图片的问题

博文 广树 2021-05-23 17:52 JavaScript

先说答案： 在需要长按保存的图片中增加一段样式 -webkit-touch-callout: default !important; 原理： 这是一段苹果Safari专用的样式，用途就是禁止图片的长按保存。估计Ionic的UI策略本来是也想禁止掉安卓的长按保存，但是安卓似乎没有对应的方案，就唯独给苹果设置上了...因为自己用的是安卓的手机，项目是明确要求长按保存的，结果一直没有发现这个问题，属实坑
