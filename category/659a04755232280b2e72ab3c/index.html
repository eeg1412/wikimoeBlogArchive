Git全局或者项目中将CRLF转为LF

博文 广树 2019-04-12 16:14 Git

之前因为WINDOWS电脑和MAC电脑交互代码发生了CRLF和LF的问题。 于是就想办法在GIT中统一。 全局： git config --global core.autocrlf false 项目中： 进入项目输入 git config --local core.autocrlf false 重新拉取下项目就会自动转为LF。
