在express-generator中使用socket.io

博文 作者：广树 时间：2019-03-27 10:31 分类：Node.js

因为socket.io官方教程都是针对express的，而express-generator中使用socket.io的资料寥寥无几。

后来参考了国外网站后，发现也挺简单的。

就是在express-generator生成的bin/www中的 

/**
 * Create HTTP server.
 */

var server = http.createServer(app);

下面添加一行 

var io = require('socket.io')(server);

然后下面的结构 

/**
 * Listen on provided port, on all network interfaces.
 */

server.listen(port);
server.on('error', onError);
server.on('listening', onListening);

io.on('connection',  (socket) => {
  socket.on('demining',  (data) => {
    console.log(data);
  });
});

就大功告成了，剩下参考官方文档即可。
