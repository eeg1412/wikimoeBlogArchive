售价高达1319多人民币的动漫小时钟！？秋叶原游记

博文 作者：广树 时间：2024-06-16 11:59 分类：周边购物游记

在结束了《孤独摇滚》联动咖啡之后，又去逛了一趟秋叶原，看看今天的秋叶原有什么新的贩卖会活动吧！


秋叶原Atre

来到秋叶原，按照惯例会看看秋叶原Atre在联动什么。

原来今天是《命运石之门》15周年的联动！

于是想进去逛一逛POP UP SHOP，结果好多人啊！按理说秋叶原Atre的POP UP SHOP队伍一般是排在楼道一路往外排的，这次居然直接在贩卖区排起来了。

无奈今天是没法逛了。


隔壁有一个《叛逆的鲁鲁修》的POP UP SHOP。

说起来，这个系列最近有一部新的剧场版，而且还是拆分成多部上映的。在我得知这个剧场版的时候已经是第二部在上映了，真感觉没什么宣传呀。


此次POP UP SHOP的商品立绘风格挺有意思的。


出门时看到了被抓起来的助手，好可爱呀！这个我记得有一套周边的。


秋葉原ホビー天国2

之后去逛了逛秋葉原ホビー天国2，这里最近在展出《魔法高校劣等生 第3季》和《街角魔族》的POP UP SHOP。


「魔法科高校劣等生 第3季」POP UP SHOP

说起来这次的第三季完全没有看呢，虽然前面2季都勉强追完了。


主题是女仆，这次主推的立绘真就完全没男主什么事情了。

另外感觉画风有些微妙不知道是为什么😅。


此次的周边商品。


「街角魔族2丁目」POP UP SHOP

然后隔壁就是《街角魔族》。

感觉最近的商业活动很多呀，那么什么时候进入3丁目呢？


全员等身看板，真的好可爱呀！


此次商品主要围绕Q版和普通比例的立绘。


T恤也很好看呀！

白色款4950日元（约228元人民币），黑色款4400日元（约203元人民币）。


不过最惊艳的，还是这些亚克力时钟。

其中左下角全员款的亚克力时钟售价高达28600日元（约1319元人民币）！

这价格比我买的那款手表都贵，我一度以为自己看错价格了，结果发现没有错！而且甚至我去的时候还卖完了！！

左上中等大小的亚克力时钟售价为7700日元（约355元人民币）。

中间迷你小时钟则是4400日元（约203元人民币）。


挂画类的周边，售价3300日元（约152元人民币）。


战利品环节

大体逛了一圈，大饱眼福，一开始其实非常想买7700日元的那款时钟，但是想了想家里已经无处安放了，所以忍住了！所以此次没有战利品。
