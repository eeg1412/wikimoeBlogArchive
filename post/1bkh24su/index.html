记录第一次在日本进行携号转网（MNP）

博文 作者：广树 时间：2024-05-18 16:20 分类：杂谈

我为什么想携号转网

早在2019年来日本工作以来，我一直使用的是UQ Mobile的3GB套餐，虽然中间因为5G的普及升级过一次套餐，但是流量一直都是3GB。

一个原因是来日本没多久的疫情，使得长期在家上班，周末也几乎没有出去玩，结果导致3GB的流量都消费不掉。

随着疫情的结束，即使开始定期出门上班了，但是由于在家和在公司都有WIFI，使用流量的场景就仅剩下了通勤的时候，此时3GB也依然消耗不完。

但是近些年，从博客中也可以看出来，我出门的机会变多了，甚至时不时会来一次说走就走的旅游，这就使得3GB的流量不够看了。

于是就想着更改手机的流量套餐，但是我所使用的UQ Mobile的下一个挡位就直接是20GB了。这显然也不是我目前能使用完的水平。

于是就开始搜索适合我的手机运营商套餐。

这不搜索不知道，一搜索发现日本的手机运营商套餐还玩的挺花的，在介绍这些特殊的套餐之前，我先来大致介绍一下日本的手机运营商是个什么状况。


关于日本的手机运营商

日本的手机运营商大致可以分为两大类MNO和MVNO。

MNO

MNO（Mobile Network Operator，移动网络运营商），是指拥有自己的通信基础设施（如基站等）并直接提供通信服务的运营商。在日本，主要的MNO包括：

  1. NTT DoCoMo
  2. KDDI (au)
  3. SoftBank
  4. 楽天モバイル

除了乐天手机（楽天モバイル）现在尚在成长，其他3个可以说是曾经垄断日本移动通信的存在。


MVNO

MVNO（Mobile Virtual Network Operator，移动虚拟网络运营商）则是通过租用MNO的网络来提供服务的运营商。

这个杂七杂八的品牌就比较丰富了，不过也大致可以分两类。

一类是上面提到的三大运营商的子品牌，另一类则是真正的租用三大运营商用剩下的带宽的，我这里就称之为野路子品牌吧。


MNO和MVNO的主要区别

最大的区别就是价格吧。

在日本三大运营商的价格是最贵的，往往只提供无限流量套餐，同时也是信号质量最好的。

三大运营商的子品牌，则是在三大运营商的基础上，针对性的增加了流量套餐等定制化内容。

不过最近子品牌们都喜欢和自家的主品牌做联动，比如用了主品牌的宽带呀，电力呀的时候给你打个7折，这就有些不打友好了。

野路子的套餐就比较花里胡哨了，不过因为是租用剩余带宽，往往网速感人。

简单总结就是：

价格：三大运营商>三大运营商子品牌>野路子品牌

通信质量：三大运营商>三大运营商子品牌>野路子品牌

这个时候就需要结合自己的实际需求选择哪一个挡位。

像我平日主要以浏览文字和图片为主，在没有视频和游戏的需求下，野路子的品牌可能比较适合我。


探寻适合自己的品牌

经过一番搜索我发现了一些非常有意思的野路子品牌。

日本通信sim

日本通信SIM是日本最早开始提供MVNO服务的公司之一。他们使用的是NTT DoCoMo的网络，并且他们的语音通话服务不是通过IP电话，而是通过一般的电话线路进行，因此可以期待清晰的音质。

他们提供了多种套餐供用户选择：

  • 290日元/月：包含1GB的数据
  • 1,390日元/月：包含10GB的数据，可以选择每月70分钟的免费通话或者每次通话前5分钟免费
  • 2,178日元/月：包含30GB的数据，可以选择每月70分钟的免费通话或者每次通话前5分钟免费

以上每个套餐都可以额外购买数据，每增加1GB需要220日元。此外，他们还提供了一些其他的服务，如国际电话、国际漫游、留守电话等。


他们家最大的特点就是便宜！对比别家是压倒性的便宜！同时也是少有在套餐内直接包含通话服务的运营商。


y.u mobile

y.u mobile是一家由U-NEXT HOLDINGS和ヤマダホールディングス共同出资设立的MVNO（移动虚拟网络运营商）。他们的目标是提供简单且满足用户需求的格安SIM服务。

以下是y.u mobile的一些主要特点和服务：

  • 永久繰り越し的数据流量：所有的数据流量都没有有效期限，可以永久繰り越。但是，数据的存储上限是100GB，超过上限的部分会从旧的数据开始消失。
  • 免费的手机修理保险：用户不需要承担任何费用，最高可以得到每年30,000日元（非税）的赔偿。
  • 包含U-NEXT的套餐：他们提供了包含U-NEXT月费的套餐，用户可以享受到U-NEXT提供的320,000本以上的视频和190本以上的杂志。
  • 优惠的iPhone：他们提供了二手的iPhone，这些iPhone的电池最大容量至少有80%，并且经过了品质检查和除菌处理。
  • 稳定的通信速度：他们的服务可以提供稳定的通信速度，使得用户可以流畅地使用社交网络和其他应用。
  • 广泛的覆盖范围：他们的服务覆盖了NTT DoCoMo的LTE网络覆盖的所有区域，包括建筑物内部、地下、山区和离岛。

他们提供了多种套餐供用户选择：

  • 1,070日元/月：包含5GB的数据
  • 2,970日元/月：包含最多20GB的数据，以及U-NEXT的月费
  • 4,170日元/月：包含最多30GB的数据，以及U-NEXT的月费，最多可以供2人使用

以上每个套餐都可以额外购买数据，每增加1GB需要330日元。此外，他们还提供了一些其他的服务，如国际电话、国际漫游、留守电话等。


他们家的主要特典是和日本的视频网站U-NEXT深度绑定，如果是使用该视频网站服务的话就会非常划算。

另外还有一个特典就是100GB以内的流量永远不会过期，可以说真正的是买多少用多少。


LinksMate

LinksMate（リンクスメイト）是由LogicLinks公司运营的一款基于NTT DoCoMo网络的MVNO（移动虚拟网络运营商）服务。它的特点是与许多在线游戏和视频流媒体服务网站进行联动，为用户提供丰富有趣的服务。

以下是LinksMate的一些主要特点和服务：

  • 游戏联动：LinksMate与许多游戏进行联动，用户可以在游戏中获得丰富的奖励。
  • 通信量计数优惠：LinksMate提供了一个名为“Count Free Option”的选项，使得特定内容的通信量计数可以减少90%以上。
  • 低速状态下的高速通信：如果用户选择了“Count Free Option”，那么即使在低速状态下，特定的通信也可以保持高速。

多种套餐：LinksMate提供了多种套餐供用户选择：

  • 165日元/月：包含100MB至1GB的数据
  • 418日元/月：包含2GB至4GB的数据
  • 858日元/月：包含5GB至9GB的数据
  • 1,518日元/月：包含10GB的数据
  • 1,738日元/月：包含12GB至18GB的数据
  • 2,618日元/月：包含20GB至80GB的数据
  • 7,953日元/月：包含100GB至1TB的数据

额外购买数据：用户可以额外购买数据，每增加1GB需要550日元。


要说上面的运营商是视频网站强化，这个运营商则是游戏强化，每个月都送游戏特典。

LogicLinks这家公司可能比较陌生，但是Cygames大家应该都知道吧，就是那个著名的游戏公司。

LogicLinks可以理解成是Cygames的子公司的存在，这就能理解为什么这个运营商做得都是和游戏相关的强化了吧！


nuroモバイル

nuroモバイル是由SONY集团运营的一款MVNO（移动虚拟网络运营商）服务。他们提供了多种套餐供用户选择：

  • NEOプラン：提供20GB的数据，月费为2,699日元。如果数据有剩余，可以翌月繰越。此外，他们还提供了Gigaプラス15GB的赠品。
  • NEOプランW：提供40GB的数据，月费为3,980日元。如果数据有剩余，可以翌月繰越。此外，他们还提供了Gigaプラス15GB的赠品。
  • VSプラン：提供3GB的数据，月费为792日元。如果数据有剩余，可以翌月繰越。
  • VMプラン：提供5GB的数据，月费为990日元。如果数据有剩余，可以翌月繰越。此外，他们还提供了Gigaプラス3GB的赠品。
  • VLプラン：提供10GB的数据，月费为1,485日元。如果数据有剩余，可以翌月繰越。此外，他们还提供了Gigaプラス6GB的赠品。
  • VLLプラン：提供15GB的数据，月费为1,790日元。如果数据有剩余，可以翌月繰越。此外，他们还提供了Gigaプラス9GB的赠品。


这家运营商要分两部分看。

一个是NEO套餐，主打的是社交媒体APP免流量，上传免流量。就很适合主播的样子。

另一个是VS/VM/VL/VLL这一块套餐，虽然没有上面套餐的免流量政策，但是主要是便宜。


我的选择

当然日本的MVNO还有很多，但是大致的候选品牌是上面描述的那几家。

在经过了一星期多的深思熟虑，我决定先试试nuroモバイル的NEO套餐。

也许就有人问了，你不是说用不完20GB的套餐吗？

因为只有他们家的NEO套餐免入网手续费3300日元和制卡费440日元。

同时使用NEO套餐8个月就返还13000日元。

(13000 + 3300 + 440) / 8 = 2092.5

差不多606.5日元用他们家的NEO套餐，所以就打算先用他们家的NEO套餐8个月，然后再看看是转他们家的5GB流量好还是10GB流量好。当然如果使用体验不好的话，也可能直接换别家的去了。


那么运营商和套餐决定好了，就开始携号转网吧！


关于日本的携号转网

首先先介绍一下日本的携号转网。

在日本，携号转网被称之为MNP（Mobile Number Portability，携帯電話番号ポータビリティ），是允许在不更改电话号码的情况下切换到其他移动运营商的一种服务。

这个过程是免费的，用户只需要从当前的运营商那里获取一个MNP预约号码，然后在新的运营商那里使用这个号码来完成切换。


申请MNP预约码

2024年5月13日早上，在上班前通过网络申请了UQ Mobile的MNP预约码。

这里不得不吐槽他们家把这个预约服务藏的真深。

在填写完一堆你为什么要换呀之类的问卷后，申请成功。

过一小会儿MNP预约号码就会以短信形式发送到手机上。


申请nuroモバイル

在获取到MNP预约号码后，直接在nuroモバイル上选择想要的套餐，然后填写自己的个人信息和MNP码，申请就完成了。

这里不得不吐槽一下这么大一家公司，邮件居然没有SSL加密！


等待审核

2024年5月15日早上收到我的审核通过，SIM卡在昨天寄出了。


拿到SIM卡

2024年5月16日，拿到SIM卡。

其实15日就送过来了，只是当时没有人收，就安排在了16日。

内容物非常简单，就是SIM卡和说明书。

我选择的是SoftBank的线路，所以他就...直接寄了张SoftBank的SIM卡😅？不愧是租用别人的线路。

总结

日本的携号转网整体体验起来非常丝滑，中间没有遇到什么坑，这点非常好评！

不过也许是因为携号转网的存在，使得大量运营商并不怎么重视老用户，转而对新用户推出非常多的优惠活动，间接导致了用户为了追求优惠，隔三岔五的进行携号转网，也挺烦的。可以说有好有坏吧。

另外虚拟运营商的存在，也一定程度的让下游运营商们的竞争越发激烈，使得市面上出现品种繁多的套餐，从而使得用户有了更多选择性。
