冒险岛日服(JMS)台服(TMS)64位客户端语言区域转区教程

博文 作者：广树 时间：2022-02-20 13:19 分类：游戏

最近，冒险岛的各个服逐渐开始普及64位客户端。这导致了原先仅支持32位的转区软件《locale emulator》不再支持冒险岛客户端转区。

下面就分享由InWILL大佬开发的支持64位软件转区软件——《Locale Remulator》

安装使用教程如下：

1.访问大佬的项目地址，找到发布地址：https://github.com/InWILL/Locale_Remulator/releases

2.找到最新发布的版本，一般在最上面。

3.下载里面的 “Locale_Remulator.x.x.x.zip”（x对应各个版本）。

例如截至发稿时间，最新版本的下载地址为：https://github.com/InWILL/Locale_Remulator/releases/download/v1.3.5/Locale_Remulator.1.3.5.zip

4.下载运行库《.NET Desktop Runtime 6.0》

下载地址为：https://dotnet.microsoft.com/en-us/download/dotnet/thank-you/runtime-desktop-6.0.1-windows-x64-installer

4.1.如果系统还缺《.NET Framework 4.8》的话，还需要下载.NET Framework 4.8，不过一般来讲大部分电脑应该是不会缺的

下载地址为：https://dotnet.microsoft.com/zh-cn/download/dotnet-framework/thank-you/net48-web-installer

5.安装第4步下载的运行库

6.解压第3步下载的Locale Remulator

7.找到register.bat文件，右键选择管理员身份运行

8.当看到如下黑色弹出框后按任意键退出即可

9.此时右键冒险岛的图标应该会出现如图菜单


10.日服使用Run in Japanese(Admin)，台服使用Run in Taiwan(Admin)

11.第一次运行可能会遇到客户端更新，在更新完后可能会出现如下弹窗

如果出现上面的弹窗，只需要点击OK重复第9到第10步（重新开一下游戏）即可。


关于更新：

如果以后出现了新的Locale Remulator版本。

我们只需要用管理员身份运行unregister.bat，然后重启再管理员运行新版本的register.bat，此时才可以删除老版本的Locale Remulator文件。
