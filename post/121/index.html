让一个项目同时提交到码云和GitHub两个仓库

博文 作者：广树 时间：2018-03-29 13:26 分类：Git

在项目目录里找到.git文件夹然后找到config文件。

打开这个文件后找到下面的代码 


[remote "origin"]
	url = git提交地址
	fetch = +refs/heads/*:refs/remotes/origin/*


将其改成 


[remote "origin"]
	url = 码云Git提交地址
	url = GitHub提交地址
	fetch = +refs/heads/*:refs/remotes/origin/*
