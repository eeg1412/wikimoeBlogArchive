看看2025年的愚人节各大动漫游戏作品都整了什么花活

博文 作者：广树 时间：2025-04-01 21:32 分类：ACG相关

又到了每年的愚人节，这次看看各大动漫游戏作品都整了什么花活吧！


《请问您今天要来点兔子吗？》

每年《点兔》都会推出愚人节特设页面，今年也不例外。

地址： https://gochiusa.com/af/

一般来讲只会愚人节当天公开，所以要抓紧时间喔！

今年的主题是《冥土 By 兔子们》，读作女仆写作冥土，不知道的还以为是万圣节活动呢。

整个流程有点像是心理测试，最终可以获得《点兔》角色的卡片，并且最终能召唤英灵🕯。


顺带一提，我的一周目就是纱路酱，好耶是纱路酱，女仆装的纱路酱好可爱呀！！！

仔细一看，似乎剧中的兔子们都变成了狗狗了呢！


人物卡片一共十人两种，都好可爱呀！

众所周知，《点兔》每年的愚人节立绘最终都会商品化，越来越期待接下来的周边了呀！


另外研究了一下游戏记录的方式，是直接往cookies里怼了 1 - 10 为 true 的cookie。

所以在控制台通过以下代码：

for (let i = 1; i <= 10; i++) {
  document.cookie = `${i}=true; path=/;`;
}

然后随便玩一周目就直接通关了😅，好孩子不要学呀！


至于最后的咒文...

这一念ko ko ro ，旋律就起来了，显然后面就是ぴょんぴょん啦！


通关后就可以收到两张海报。


《我推的孩子》

今年继续迫害加奈。

大致的内容是：

时隔数年，青椒再度席卷社会！？
有马加奈的活跃止不住！
“宫崎县产青椒‘大使’！？”
（有马加奈成为宫崎县产青椒‘大使’！？）

人气女演员、B小町成员有马加奈除了演艺事业之外，又迎来了新的活动方向。她将于4月1日被任命为宫崎县青椒“Green Sunrise”的“青椒大使”。

“青椒体操”也让世界瞩目
有马加奈与青椒的渊源颇深，甚至可以追溯到她的童年时期。当时她因青椒讨厌症而痛苦不堪，并在电视节目中挑战过“青椒体操”，最终以一世轰动的形式被世人所知。这段经历让她在音乐界获得大量支持，甚至音乐排行榜上也占据一席之地。
如今，她正式成为“青椒大使”，不仅是在宣传农业，还计划进行更大规模的推广活动。对于这一任命，她表示：“当接到邀请时，我一度很慌张，但最后决定认真执行。”

大热MV的共通点
最近B小町的MV《POP IN 2》在宫崎县拍摄，而巧合的是，这正是她将担任“青椒大使”的地方。青椒大使的“Green Sunrise”品牌与她的形象契合度极高，因此MV的视觉效果也与此形成强烈的共鸣。

“今年是青椒大使之年！？”
（今年，竟然是“青椒大使”就任的一年！？）

（注：本文为愚人节特辑，纯属虚构。）
