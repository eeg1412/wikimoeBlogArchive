历经多重考验的第一天——《摇曳露营△》圣地巡礼之伊豆高原大室山和仙人掌动物公园

博文 作者：广树 时间：2024-10-15 20:03 分类：圣地巡礼

2024年10月13日，天气晴，今天出发去伊豆进行为期两天的《摇曳露营△》圣地巡礼。

本次圣地巡礼主要是去《摇曳露营△》第二季，抚子等人的伊豆之旅中出现的场景，第一天的行程是去位于伊豆的伊豆高原附近的大室山和仙人掌公园。


启程去伊豆高原

从东京去伊豆高原的大室山和仙人掌动物公园的方法有两种。

  1. 在东京站乘坐特急舞女号经过伊东来到伊豆高原，最后座巴士到仙人掌公园站，需要大约2.5小时的车程。
  2. 在东京乘坐新干线到热海站，再从热海站乘坐伊东线到伊东站，最后在伊东站乘坐巴士到仙人掌公园站，需要大约2小时的车程。

我这次是选择了第一种方案。

通过网络可以提前购买去伊豆高原的特急车票，照片中展示的仅仅是乘车券，还有一张特急券我选择了电子版。

乘车券加特急券原价为5170日元（约247人民币）。

座位方面，如果想看大海的话推荐A侧，但是代价是，巨晒！

虽然不是全程看海，但是如果喜欢看海的话推荐A侧。

顺带一提D侧则是看山。


来到伊豆高原

大约2小时的车程来到了伊豆高原站。


第一难——乘坐公交去仙人掌公园车站

来到伊豆高原后第一时间确认公交车站台的位置，并迅速加入排队的队伍之中，因为这一车下来的人是九八九都是冲着大室山去的，而公交车的班次非常少一车又送不完一批人，就有可能导致经历漫长的等公交时间。

注意目标站名是仙人掌公园车站（シャボテン公園），因为大室山和仙人掌动物公园是紧挨着的。

此时也可以选择出租车，不过也要第一时间冲上去，因为这个时候出租车站台也会排满人😅，而伊豆整个区域的出租车资源非常紧张，什么时候能排到就看运气了。

然后这一天，运气非常差，遇到了因为交通事故导致的大堵车！

导致本来20分钟的公交车车程，花了1小时20分钟才到达目的地😩。

太难了，真的太难了，因为没坐上公交车的座位，而公交车又塞满了人，导致在公交车上一直是一个扭曲的身姿，更加糟糕的是，因为公交车上人太多，导致手机进入了有信号但是没有网络速度的情况😱。


来到仙人掌公园车站

就这样，以一种扭曲的姿势，罚站了1小时20分钟，终于可以车了。

看了眼眼前的情况...

这是什么长龙？

因为《你的名字》的大火和小红书的推波助澜，导致大室山成为了一个网红打卡圣地。

于是来自国内的旅游团络绎不绝，导致本来就薄弱的伊豆交通雪上加霜。

而去大室山是不能步行上山，必须通过缆车，所以这一系列原因导致了大室山的超长排队时间。

本想着国庆节都结束了应该会好一些吧，看来还是我想得太美了😭。


仙人掌动物公园

由于大室山那边的队伍太过夸张，于是决定先逛仙人掌动物公园。

因为是从大室山到动物公园，所以少了很多正门的场景，有点可惜。

仙人掌动物公园的门票为大人2800日元（约134人民币）。


卡皮巴拉彩虹广场

在这个动物公园里，最明星的动物就是卡皮巴拉（水豚）了。

在卡皮巴拉彩虹广场，可以零距离触摸卡皮巴拉。

和动画相比，现实世界是人比卡皮巴拉多系列。

到底是人在看卡皮巴拉还是卡皮巴拉在看人，这是一个问题。


不过确实卡皮巴拉的性格很温和，也比我想象的要大只。


卡皮巴拉图腾。


路上见到的其他动物

一路上动物种类还是很丰富的。


很多动物都是纯放养状态，难怪动画里老师和售票员说路上看到很多动物之后，售票员那么慌😅。感觉确实很容易逃出去呢。


路上见到的各种建筑雕像

动物园里各种古文明的雕像也挺多的。


鸟类天堂

这里鸟类品质也很多，也是放养状态的。



可以零距离观看各种鸟类。


在动画里出现的鲸头鹳，很可惜因为寿命的原因，已经去了另一个世界。

动物公园做了很多纪念这位长寿鲸头鹳的内容。

并成为了第二代动物园名誉园长。


太阳广场

正门的一个大广场，因为临近万圣节，摆放了巨型南瓜。

背后则是大室山。


卡皮巴拉露天澡堂

因为不是冬天的原因，没有卡皮巴拉乐意泡澡😅。

而且现实中的池子有点脏呀，甚至还有粑粑。


小熊猫馆

小熊猫真的太可爱啦！

这里可以花钱拿上设备去喂小熊猫。


高原龙温室

在进入前就好奇这些金字塔是什么，原来是培养仙人掌的温室。

确实既然叫仙人掌动物公园，怎么能少仙人掌。


龙看上去好气派！

不过上面停了好多鸟😅。


有好几个区域，里面的仙人掌种类非常繁多，最后甚至可以当场买一个自己心仪的仙人掌。


在仙人掌温室里，同时还展出了许多品质的猫头鹰。

大早上的，估计都在半睡眠状态吧。


购物区

除了仙人掌温室，就是喜闻乐见的购物区了。

购物区自然少不了各式各样的卡皮巴拉玩偶。


午饭时间

说起来因为大堵车的原因午饭都还没吃呢，买个特色仙人掌果实冰淇淋充充饥。

虽然完全没吃过仙人掌果实，但是感觉不出什么特别的味道😅。


大室山

在逛完仙人掌动物公园，接下来就是去看看大室山的排队状况了。

这动画中的场景根本就不可能出现在现实中嘛！


第二难——缆车上山

一开始看缆车的营业时间是到16点，担心如果超过16点是不是就白排了。

后来发现其实只要在16点前排在队伍里了，那么最终就会负责把你送上山和送下山。

这次的缆车排队时长一共花了足足2小时！

缆车票价1000日元（约48人民币），包上下山。



上山时，脚是浮空的，所以千万记着绑紧鞋带。


中途会有一个拍照点，到山顶后会询问要不要买，一张1200日元（57人民币）。


山顶

动画中的标志性场景。

可惜，这一天能见度不行，看不到富士山。


确实还是要赶绿色才好看！


前面提到的《你的名字》中出现的场景。

个人觉得也不是那么像呀，怎么就火了？


正逢夕阳，风景确实很美，就是有一些雾气。


动画中出现的地藏。


第三难——缆车下山

兜兜转转半小时，差不多准备下山了，要不然赶不上17点30分前往伊豆高原车站的巴士了。

于是第三难开始了。

下山高峰...

也是，上山排队多久，那么下山肯定也要花相对应的时间。


就这样又排了好久，天都黑了，终于坐上了缆车下山。

天黑了，风也大了，摇摇晃晃的还挺带劲的😅。


回程

从大室山下来之后看了看时间，17点40分！完美错过了末班车！


第四难——返回伊豆高原车站

第四难来了，我要怎么返回伊豆高原车站呢？

于是我打开了约车APP，显示...当前没有可用出租车！？

于是换了好几款APP都是这么显示的...

打出租车公司电话，忙音...

这下麻烦了😱，要怎么回去？？？

于是问了问周围一样回不去的难兄难弟们怎么办，最后达成一致，走路回去！

走路回去需要大约1小时，而且都是下坡😱。

没办法只能硬着头皮了。

好在人多，问题应该不大？

在步行过程中，发现不仅有些路段坡道急，而且很多路段是一点路灯都没有。

只能借助月光和手机的灯光，勉强判断路面情况。

于是就这样，一群人走在伸手不见五指的山路，艰难的前往伊豆高原车站。

步行大约1小时候，我们居然真的成功到了车站！当时那个成就感，至今难以忘怀。以后一定会成为一段美好回忆吧。


来到伊豆高原车站，此时已经接近19点，没想到中午那么热闹的车站这个点居然空无一人。

来回的车辆也几乎是空载状态。

看来对于伊豆这个地方来说，除了游客，本身利用公共交通的人并不多，而且游客都是集中在某一个时间段扎堆出现，这就对当地的公共交通带来了严峻的考验。


最后在电车上和战友们一一告别之后，成功回到了位于伊豆下田的酒店。

历经多重考验的第一天——《摇曳露营△》圣地巡礼之伊豆高原之旅就结束了。

明天继续《摇曳露营△》的圣地巡礼，敬请期待！
